# MALytics — Сервіс персональних рекомендацій аніме

MALytics — це веб-додаток, який аналізує профіль користувача на MyAnimeList і пропонує нові тайтли для перегляду. Система порівнює ваші вподобання з глобальним рейтингом, враховуючи переглянуті списки та жанрові пріоритети.

## Функціональні можливості

* Аналіз профілю: Визначення улюблених жанрів на основі оцінок користувача.
* Розумний підбір: Порівняння інтересів з Топ-1000 аніме (MAL API).
* Локалізація синопсисів: Автоматичний переклад описів аніме на українську мову.

## Технології

* Бекенд: Node.js, Express.
* Фронтенд: HTML, CSS (Variables), Vanilla JavaScript.
* Іконки: Phosphor Icons.
* Робота з даними: MyAnimeList API, Google Translate API (для локалізації).

## Встановлення та запуск

### Попередні вимоги
* Node.js (версія 14 або вище).
* MyAnimeList Client ID (реєструється в налаштуваннях профілю MAL).

### Кроки для запуску
1. Клонувати репозиторій:

   git clone https://github.com/Fitil4ik/MALytics.git

   cd MALytics

2. Встановити залежності:

   npm install

3. Створити файл .env у кореневій директорії та додати свій ID:

   MAL_CLIENT_ID=ваш_ідентифікатор

4. Запустити сервер:

   npm start

Після запуску додаток буде доступний за адресою http://127.0.0.1:8000.
